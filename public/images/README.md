# Game Images Setup

This directory should contain the coupon images for each game.

## Required Files

### 1. expedition33.jpg

- **Game**: Clair Obscur: Expedition 33
- **Recommended size**: 800x600px or similar aspect ratio
- **Style**: Dark fantasy, chiaroscuro aesthetic
- Place this file in `/public/images/expedition33.jpg`

### 2. baldursgate3.jpg

- **Game**: Baldur's Gate 3
- **Recommended size**: 800x600px or similar aspect ratio
- **Style**: Fantasy RPG, D&D aesthetic
- Place this file in `/public/images/baldursgate3.jpg`

## Image Guidelines

- Format: JPG or PNG
- The images will be displayed with 30% opacity as background on the coupon front
- High-quality game artwork or promotional images work best
- Ensure you have rights to use the images

## Fallback

If images are not provided, the coupons will display without background images (icons only).
