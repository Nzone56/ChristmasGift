# Audio Files Setup

This directory should contain the theme music for each game.

## Required Files

### 1. alicia.mp3

- **Game**: Clair Obscur: Expedition 33
- **Track**: "Alicia" by Lorien Testard
- **Source**: Point'n Think
- Place this file in `/public/audio/alicia.mp3`

### 2. down-by-the-river.mp3

- **Game**: Baldur's Gate 3
- **Track**: "Down by the River" by Borislav Slavov
- **Source**: Official Baldur's Gate 3 Soundtrack
- Place this file in `/public/audio/down-by-the-river.mp3`

## Audio Format

- Format: MP3
- Recommended bitrate: 192kbps or higher
- The audio will loop continuously while the theme is active
- Volume is set to 40% by default with smooth fade transitions
